Berechnung des Wirkungsquerschnitts

GSI – Daten

Zur Berechnung des Wirkungsquerschnitts der GSI Daten wird ein vorhandenes Skript von Dr.-Ing. Hannes Zöllner verwendet. Um das Skript verwenden zu können muss das Panorama in eine csv Datei umgewandelt werden.

Dazu wird das Bild eingelesen und Pixelweise ausgewertet. Wenn das Pixel rot ist, wird an seiner Position die Impulsenergie der Goldionen (welche) in eine Liste eingetragen, ansonsten eine 0. Diese Liste wird anschließend als csv-Datei gespeichert.
